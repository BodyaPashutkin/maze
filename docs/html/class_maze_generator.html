Detailed Description

Класс для генерации лабиринтов и поиска путей.

Реализует генерацию идеального лабиринта с помощью алгоритма DFS с бэктрекингом, а также поиск кратчайшего пути с помощью BFS. Вся память управляется автоматически (std::vector).
