# Maze Generator

Проект представляет собой консольное приложение на C++ для генерации идеальных лабиринтов и поиска выхода из них. Разработано в рамках курса "Алгоритмизация и программирование".

## Назначение программы и функционал
Программа позволяет генерировать случайные лабиринты произвольного размера и предоставляет интерактивный режим для их прохождения.

**Основные возможности:**
* Генерация идеального лабиринта с помощью алгоритма поиска в глубину (DFS) с бэктрекингом.
* Поиск кратчайшего пути от текущей позиции до финиша с помощью алгоритма поиска в ширину (BFS).
* Интерактивный консольный интерфейс (управление WASD, подсказка пути по кнопке H).
* Сохранение сгенерированного лабиринта в текстовом формате и загрузка из файла.
